# DUT0BUG-RM-CV-2021

## 项目简介
本仓库为2021年大连理工大学凌BUG战队manifold电控组代码
### 开发环境
```
DJI Manifold 2-G, Ubuntu 16.04
迈德威视工业相机SDK, OpenCV+OpenCV_contrib 4.5.2, Qt 5
```
### 开发者
王老师和他的工具人们
